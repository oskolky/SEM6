SynParser — User Guide

What is SynParser?

SynParser performs syntactic + semantic analysis of English text using spaCy, served via a FastAPI backend.

  • 🏷️ POS tagging — part of speech for every token
  • 🌲 Dependency parsing — syntactic head-child tree
  • 🏛️ Named Entity Recognition — people, orgs, places, dates…
  • 🎬 Semantic Role Labelling — who did what to whom
  • 📝 Lemmatization + Morphology
Quick startPaste text → ▶ Analyse → switch to Semantics tab.

Constituency Tree

The app builds a phrase-structure tree for each sentence (S, NP, VP, PP, ADVP) and renders it as a top-down SVG graph in the Semantics tab.

Named Entity Recognition (NER)

Entities are highlighted inline in the text. Types include:

PERSONPeople, including fictional
ORGCompanies, agencies, institutions
GPECountries, cities, states
LOCOther locations, bodies of water
DATE / TIMETemporal expressions

PRODUCTObjects, vehicles, foods
EVENTWars, battles, sports events

Semantic Role Labelling (SRL)

For each verbal predicate a frame is extracted showing:

  • Agent — who performs the action (nsubj)
  • Patient — who/what receives it (dobj/obj)
  • Location — where (in/at/on + place)
  • Time — when (temporal phrases)
  • Instrument — with/by what means
  • Purpose — for what goal
  • Manner — how (adverbs)
NoteSRL is heuristic (dependency-based), not a full PropBank model. For production use consider adding spacy-transformers.
