SynParser — User Guide

What is SynParser?

SynParser performs automatic syntactic analysis of English text using the spaCy NLP library, served through a FastAPI backend.

For each sentence it computes:

  • 🏷️ Part-of-Speech (POS) tags for every token
  • 📝 Lemmas — base (dictionary) forms of words
  • 🌲 Dependency tree — syntactic head-child relations
  • 🔍 Key roles — subject, predicate, object, modifiers
  • 🧬 Morphology — tense, number, case, etc.
Quick start Paste English text into the left panel → click ▶ Analyse → explore results in the tabs on the right.

Input Text

Paste any English text directly into the textarea. The counter shows current length (max 50 000 characters).

File Upload

Drag & drop a file onto the dashed zone, or click it to browse. Supported formats:

TXTPlain text
PDFVia pdfplumber / pypdf
HTMLVia BeautifulSoup
RTFVia striprtf
DOCVia python-docx
DOCXVia python-docx
Note File size limit is 10 MB. Text is extracted automatically before analysis.

Save Results

Click 💾 Save to download the full analysis as analysis.json — contains all sentences, tokens, trees and roles.

Result Tabs

Summary
Coloured token strip with POS labels + key syntactic roles (Subject / Predicate / Object / Modifiers). Hover any token chip to see its lemma, dep label, head and morphology.
Tokens
Full token-by-token table: index, text, lemma, POS, fine-grained tag, dependency role, head word, stop-word flag, morphological features.
Dep Tree
Interactive SVG dependency tree. Arcs go from dependent to head. The ROOT token is highlighted in black. Scroll horizontally for long sentences.
JSON
Full syntax-highlighted JSON of the current sentence. Click Copy to put it in the clipboard for use in downstream scripts or APIs.
Sentence navigation Use the #1 #2 … pills above the tabs to switch between sentences. Arrow keys ← → also work.
