📖 Help & Documentation

Getting Started

1. Go to Upload and add a text file to your corpus (TXT, PDF, DOCX, or RTF).

2. Add metadata: title, author, genre, year, source — this helps filtering later.

3. After upload, the text is automatically processed by spaCy NLP: tokenized, lemmatized, POS-tagged.

Frequency Analysis

Wordforms — raw surface forms as they appear in the text.

Lemmas — base/dictionary forms (run = ran = runs).

POS distribution — how many nouns, verbs, adjectives, etc.

Type/Token Ratio (TTR) — vocabulary richness (0–1). Higher = more diverse.

Concordance (KWIC)

KWIC = Key Word In Context. Shows every occurrence of a word with surrounding context.

Search matches both the exact wordform and its lemma (so searching "run" also finds "ran", "runs").

Use the window slider to control how many tokens appear on each side.

Morphology

Look up any word to see all its grammatical forms found in the corpus: POS tag, morphological features, dependency role, and frequency.
